# JogoDaForcaV2



![](https://imgur.com/dIElNTD.gif)

## Introdução

O Jogo da Forca é um clássico dos desafios de palavras, divertido e educativo.
O objetivo é adivinhar uma palavra ou frase oculta, letra por letra, antes que o desenho de um boneco seja completado. Cada erro adiciona uma parte ao boneco, e quando ele está inteiro, o jogador perde.



## Funcionalidades

- O computador irá escolher de forma aleátoria um palavra em um banco de dados(Animais,paises e frutas)
- Há um limete de 5 tentivas depois disso ele perde
- Menu de escolhas de qual lista quer prosseguir
- Armazenamento de letras já chutadas
- Tem a possiblidade do usuario digitar uma palavra inteira
